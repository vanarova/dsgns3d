# Summary

This is model is now for sale as an Olympique Lyonnais Lion replica in our Shapeways shop, it has been scaled and hollowed to save material price: https://www.shapeways.com/product/JHH2U9FZ8/olympique-lyonnais-lion-statue?optionId=60558918&li=marketplace

I found a model of a lion (kind of low poly) that needed fixing so I used the free program MeshMixer (http://www.meshmixer.com) to fill holes and scale/orient it.  
The original file is here: http://www.archibaseplanet.com/download/ce7e3456.html
